# reaktive

My monolith spring boot code base.

A web application based on: 
* jvm
* gradle
* kotlin-coroutines
* spring-webflux
* spring-security
* spring-r2dbc
* H2database

**to test the app**<br/>
open terminal and run `./gradlew check`<br/>

**to run the app**<br/>
`./gradlew` and launch http://localhost:8080/
